# Air-Writing-AI-Project-for-Capstone
Air Writing AI Project for Capstone

# 🖐️ 차세대 AI 비전 기반 가상 입력 장치 (Virtual Vision Input Device)
단국대학교 컴퓨터공학과 캡스톤 디자인 프로젝트

## 📌 프로젝트 개요
본 프로젝트는 라즈베리파이 5와 AI HAT+(Hailo-8L)를 활용하여, 허공에서의 손동작을 인식해 마우스 제어 및 텍스트 입력을 수행하는 엣지(Edge) 기반 차세대 입력 장치 개발을 목표로 합니다. 별도의 물리적 장비 없이 카메라 한 대로 사용자의 제스처를 추적하여 직관적인 컴퓨팅 환경을 제공합니다.

## 💡 프로젝트 선정 배경 및 장점
* **엣지 AI 하드웨어 성능 극대화:** 라즈베리파이 자체 CPU만으로는 30fps 이상의 실시간 랜드마크 추적이 버거울 수 있으나, 13 TOPS 성능의 Hailo-8L NPU를 활용하여 비전 처리와 AI 추론을 지연(Lag) 없이 매끄럽게 수행합니다.
* **명확한 파이프라인과 역할 분담:** 임베디드/비전 제어 파트와 딥러닝 모델링 파트로 역할이 명확히 나뉘어, 2인 팀 프로젝트로서 최적의 효율을 낼 수 있는 구조입니다.

## ✨ 핵심 기능 및 문제 해결 알고리즘

### 1. 직관적인 모드 전환 (State Machine)
마우스 기능과 필기 기능의 충돌을 방지하기 위해, 제스처 기반의 상태 머신(State Machine) 구조를 도입했습니다.
* ✌️ **V자 (검지, 중지 폄):** 가상 마우스 모드 진입
* ☝️ **검지만 폄:** 공중 필기 모드 진입
* 🖐️ **손바닥 쫙 폄:** 초기화 (ESC / 캔버스 지우개 역할)

### 2. 가상 마우스 (Virtual Mouse)
카메라에 잡힌 손가락 끝(Landmark) 좌표를 모니터 화면의 해상도에 매핑하여 마우스를 제어합니다.
* **기술 스택:** MediaPipe Hands (관절 추적) + PyAutoGUI (OS 제어)
* **클릭 구현:** 엄지와 검지가 맞닿는 제스처(Pinch) 인식 시 좌클릭 이벤트 발생
* **사용자 경험 개선:** 허공 체공 시 발생하는 미세한 손떨림을 방지하기 위해, 이전/현재 프레임 좌표의 평균을 내는 **이동 평균 필터(Moving Average Filter)**를 적용하여 커서 이동을 부드럽게 최적화했습니다.

### 3. 공중 필기 및 문자 인식 (Air-Writing & OCR)
허공에 쓴 글씨(A\~Z, 0\~9)를 인식하여 텍스트로 입력하는 기능입니다.
* **Pen-Up / Pen-Down 문제 해결:** Pinch 제스처를 유지하며 이동할 때만 가상 캔버스에 하얀 궤적이 렌더링되도록 트리거를 설정하여, 단순 손 이동과 필기 궤적을 완벽히 분리했습니다.
* **데이터 생성 현실화:** 수천 개의 공중 필기 데이터를 직접 구축하는 대신, 완성된 궤적을 2D 이미지로 변환한 후 기존의 방대한 **EMNIST 데이터셋**으로 학습된 모델에 통과시키는 방식으로 우회하여 개발 효율성을 극대화했습니다. (한글의 복잡성을 고려, 1차 목표는 영어 알파벳과 숫자로 한정)
* **NPU 실시간 추론:** 경량화된 CNN 모델(MobileNet 등)을 PC에서 학습한 후, Hailo 포맷(`.hef`)으로 양자화(Quantization)하여 AI HAT에 적재, 0.01초 이내의 빠른 인식 속도를 달성합니다.

## 🛠 하드웨어 및 개발 환경

### Edge Device (Inference & Control)
* **SBC:** Raspberry Pi 5
* **NPU:** Raspberry Pi AI HAT+ (Hailo-8L, 13 TOPS)
* **Camera:** Raspberry Pi Camera Module 3 Wide (11.9MP, Sony IMX708, FOV 120°)
* **Display:** 포터블 모니터 (시연 및 실시간 UI 출력용)

### AI Workstation (Model Training)
* **GPU:** NVIDIA GeForce RTX 4080 Super
* **RAM:** 64GB
* **Storage:** 4TB (대용량 EMNIST 데이터셋 및 모델 체크포인트 관리)

  * 궤적 이미지 전처리 및 모델 추론 파이프라인 구축
